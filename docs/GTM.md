# Go-To-Market Notes

## Positioning

StackSave is positioned as an AI spend audit for startups that want to reduce
overspending on tools, seats, and overlapping subscriptions.

## Core Message

The product helps teams answer three questions quickly:

- what are we overspending on
- how much can we save
- what should we change first

## Primary Audience

- startup founders
- operations leaders
- finance-minded team leads
- technical managers responsible for AI tooling budgets

## Sales Motion

The current product supports a lightweight self-serve motion:

1. User lands on the marketing page.
2. User completes the audit form.
3. User reviews the savings report.
4. User submits a lead form for the full report or follow-up.

## Differentiation

- deterministic savings logic instead of vague AI claims
- shareable public report for internal forwarding
- fast audit flow that does not require setup or a billing export
- MVP scope that is easy to demo in an internship review

## Next GTM Ideas

- founder-focused landing page variants
- stronger call-to-action placement near the report summary
- benchmark-style messaging for common AI stack waste patterns
- outbound messaging tailored to small teams with high per-seat tool spend
