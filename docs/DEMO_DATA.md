# Demo Data

Use this scenario for screenshots, demos, and recruiter walkthroughs.

## Scenario

Paul Industries is a 35-person startup using several AI tools across product,
engineering, and research. The company has grown quickly and now has overlapping
subscriptions, too many seats, and a few plans that are larger than the current
team needs.

## Expected Story

- ChatGPT and Claude create general-assistant overlap.
- GitHub Copilot and Cursor create coding-assistant overlap.
- Enterprise plans are questionable for a 35-person team.
- Seat counts above team size should produce cleanup opportunities.
- The report should show enough savings to trigger the Credex consultation CTA.
